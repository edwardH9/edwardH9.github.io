Security System Project

Overview

This project aims to make an affordable security camera system

How the system works

The ESP32 Cam is a microcontroller that has WiFi capabilties and an OV2640 camera to record video and capture still images. The code for this module, relies on these two fundamental properties of the device. The ESP32 Cam first begins recording upon being powered on. Then the WiFi chip begins to scan for networks in the area. Once it finds a recognized network, the ESP32 Cam connects to the network and streams the image to a specific IP address in that network. The user can then use any computer in the same network, and view the live stream on any monitor of choice by accessing that IP.

I also added solar power circuitry to this project. This circuitry has necessary battery protections and will allow the cameras to operate 24/7 without the need to take out and recharge the batteries. The main battery is a set of 18650 Lithium Ion batteries coupled to a Lithium battery solar charger IC. In times of sufficient sunlight, this panel is capable of bypassing the battery and powering the camera directly thus improving battery longevity.
