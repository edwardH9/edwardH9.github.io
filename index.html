NASA Project: Autonomous Drone/Rover Hybrid Vehicle

In the Summer of 2021, I lead a team of 3 students in designing, constructing, and programming a standalone autonomous vehicle capable of traveling through land and air.


This project also has an integrated Webserver, making it a full stack project.

This project uses Python and C++ for the robotic control systems, and I developed a server for this unit using Node.js, Express.js, EJS, and MySQL.

Background

I was selected among 7 other students to represent my college in an internship with NASA in 2021. My vision for this program was to use the Raspberry Pi SBC and peripheral microcontrollers to make a vehicle that could travel through both air and land.
