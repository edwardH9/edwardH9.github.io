Using AI to Predict Employee Turnover

This machine learning model takes in an employees information including age, occupational industry, work experience, and other factors to determine if the employee is likely to leave their job. This model was created in TensorFlow and reaches 90% validation accuracy across 15 professions and 16 industries. Training data is preprocessed such that all categories are normalized and any new input is also automatically normalized.

Virtual Assistant AI

This project uses 1 image generation model, 3 NLP models, 1 NER model, and serialized memory to function as a personal assistant. The image generation models create different faces for the AI's avatar, and the NLP models determine the users emotion through their sentences, as well as interpret commands the user gives. The NER model predicts important people or locations to the user, and stores these in memory at the users request. All models are created in PyTorch and all data work is done in Python using Pandas and SciKitLearn.
