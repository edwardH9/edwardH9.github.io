Liquid Rocket Control Systems at UC Davis

Complete control system design of rocket engine developed at UC Davis under my lead

My role with the rocket engine is to create a system that gathers high fidelity & high speed data on the state of the engine while still being able to process external commands.
Technologies Involved:

  1. Multithreaded Programming
  2. Serial Communication via SPI & UART
  3. Network Communication via UDP
  4. Signal Processing

Pictured above is the full system layout. The heart of the system is an Analog to Digital converter that maps the voltage from each sensor to a digital signal which the main processor reads. To log the data, the main processor sends the data to the main computer along with a timestamp. To maximize efficiency, we split the cores of the processor in software such that one core handles processing and transfer from the main ADC, and the other core receives commands and performs corresponding actions including actuating valves, throttling the engine, etc.
