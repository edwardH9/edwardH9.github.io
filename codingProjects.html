Wind Power Visualization Project

In this project, I updated two existing web visualization programs with new Wind Power generation data from power plants across California.

This project consists of two different visualizations. One visualization is based on each plants capacity factor, which is a ratio comparing the electricity generation that the plant is outputting relative to its maximum possible output. The second visualization shows the monthly change in the capacity factor, for each power plant during the years 2014-2020. This project spans over 10,000 lines of code and the end product is displayed on the CEC's official website.

Server with Secure MySQL Database

This server is half of my final project for the NASA Microcomputers and Robotics internship I have been involved in from Summer 2021 to Spring 2022.

This server is built using Node.js, Express.js, EJS, and MySQL and is part of a rover vehicle I created. The server hosts data that the rover collects while it is deployed, such as ambient temperature, mission uptime, deployment location, and information on the mission's operators.
