Generalized Artificial Intelligence

The goal of this project is to create an artifical intelligence unit that can mimic human interaction. The unit so far consists of 3 text-based models and 2 image-based models, with the following abilities:

  1. Distinction of sentences into categories: Question, Statement, and Command
  2. Classification of emotion along 26 categories
  3. Text generation with custom vocabulary
  4. Generation of custom avatars
  5. Short and long term adaptation to user based on model outputs
The model architectures used include RNN's, GAN's, Encoder-Decoder with Attention, and simple linear. All image models are trained using augmented data.

Technologies Used:
  1. Anaconda Virtual Environment
  2. Python
Python Libraries Used
  1. Pytorch
  2. Panda's, Numpy, Matplotlib, etc.

The main use case for the unit is as a personal assistant. Thus, the unit can access websites for tasks such as reporting daily news events, reading scientific journals, checking weather forecasts, etc. The unit actively builds and maintains memory of common people/things that the user mentions, and persists its memory in a serialized data format between interactions. The unit speaks to the user if possible, and otherwise prints its output to the terminal.

When a user first runs the program, the program will generate a human face and ask the user if the user approves of the generated avatar. The program will continually generate new faces until the user approves, in which case the program will save that face as part of the programs main object.

After this, the program asks the user if they would like the program to perform preset start up functions. By default, these include stating the date and time, reading out the top news headlines of the day, reporting recent experiments in physics, and read out recent emails.

Sample output of unit performing basic read-out functions:

After this, the unit will enter conversational mode. In this mode, the program will wait for the user to speak. When the user speaks, the program will classify the sentence and respond conversationally to plain statements, or if the sentence was a command, the program will execute the command. The program will in either case also determine the emotion of the users sentence. In future versions, the responses that the program gives will take this sentiment into account.
